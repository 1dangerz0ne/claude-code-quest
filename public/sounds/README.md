# Sound Effects for Claude Code Quest

This folder should contain the following sound files:

| File | Purpose | Suggested Source |
|------|---------|------------------|
| `tap.mp3` | Button clicks | Short click sound |
| `correct.mp3` | Correct answer | Bright chime/ding |
| `wrong.mp3` | Wrong answer | Low buzz/error tone |
| `combo.mp3` | Combo milestone | Rising tone |
| `levelup.mp3` | Level up | Fanfare/achievement |
| `streak.mp3` | Daily streak | Success jingle |

## Free Sound Sources

1. **Pixabay Sounds** (No attribution required)
   - https://pixabay.com/sound-effects/

2. **Freesound.org** (Requires free account)
   - https://freesound.org/

3. **Mixkit** (Free with attribution)
   - https://mixkit.co/free-sound-effects/

## Recommended Sounds from Pixabay

- Tap: Search "ui click" or "button click"
- Correct: Search "success" or "positive notification"
- Wrong: Search "error" or "negative notification"
- Combo: Search "level up" or "power up"
- Level up: Search "achievement" or "fanfare"

## Technical Notes

- Keep files under 50KB each for fast loading
- MP3 format is widely supported
- Volume normalization helps consistency
- The SoundManager preloads all sounds on first user interaction
